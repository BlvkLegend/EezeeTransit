# EezeeTransit

A modern logistics and transportation website for Eezee Transit, showcasing courier delivery, freight services, vehicle rentals, and bus charter solutions through a responsive, conversion-focused design.

## Live Demo

🔗 https://blvklegend.github.io/EezeeTransit/

## Features

- Responsive multi-page design
- Courier and logistics service showcase
- Vehicle rental and bus charter information
- Modern UI with smooth interactions
- Mobile-first layout
- Fast-loading static website
- Built with HTML, CSS, and JavaScript
